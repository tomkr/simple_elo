Simple elo

A simple gem for calculating elo ratings.

View on GitHub Download .zip Download .tar.gz

Welcome to SimpleElo

It really is simple:


  player1 = {rating: 1500.0, k_factor: 20}
  player2 = {rating: 1500.0, k_factor: 20}
  SimpleElo.calculate_match(player1, player2, 1)
  => [1510.0, 1490.0]

SimpleElo was developed by @tomkr for the @brightin foosball app.
